## Notes on Citations

1. **Primary Source**: The Anthropic Threat Intelligence report is the primary source for all information about the GTG-1002 campaign.

2. **Secondary Sources**: Security solution information is based on publicly available product documentation and should be verified with vendors.

3. **Case Study Scenarios**: The LogiCorp International scenario is a composite case study designed to illustrate typical organizational responses. It is not based on a specific real-world incident.

4. **Personal Communications**: Any references to personal communications (e.g., "LogiCorp Security Team, personal communication") are fictional and used for narrative purposes only.

## Updating Citations

When adding new sources:
1. Add complete citation information to this document
2. Use APA style format
3. Include URL if available online
4. Update in-text citations in relevant documents
5. Maintain consistency across all documents
