## Document Standards

### Citation Style
- **Format**: APA Style (7th Edition)
- **In-text**: (Author, Year)
- **References**: Complete reference list at end of documents

### Writing Style
- **Technical Documents**: Formal, technical language
- **Case Studies**: Narrative, accessible language
- **Balance**: Technical accuracy with accessibility

### Formatting
- **Markdown**: Standard Markdown syntax
- **Visualizations**: Mermaid diagrams where they add value
- **Tables**: Used for structured data
- **Headings**: Clear hierarchical structure
